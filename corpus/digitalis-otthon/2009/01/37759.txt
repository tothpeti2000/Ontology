Aki az év második felében vásárol Vistát, kedvezménnyel frissíthet Windows 7-re

A több Windows-változat érkezésének idõpontját korábban napra pontosan megjósoló TechARP.com úgy tudja, 2009. július 1. után a Microsoft lehetõvé teszi a PC-gyártók számára, hogy a Windows Vistával értékesített számítógépeik mellé egy olyan kupont csomagoljanak, amely a Windows 7 kedvezményes, vagy akár ingyenes beszerzésére jogosít.



2006 októberében, a Windows Vista megjelenése elõtt hasonló akciót indított már a Microsoft, akkor a Windows XP-vel eladott gépek tulajdonosai voltak jogosultak kedvezményes Vista-kuponra az Express Upgrade program keretében. A PC-gyártók maguk dönthették el -- bizonyos keretek között --, hogy mennyit kérnek a frissítésért, így eshetett, hogy míg a Dell vásárlóki 49 dollárt fizettek a Vista-kuponért, addig a HP vevõi ingyen kapták. A TechARP.com úgy tudja, a Windows 7 esetében sem lesz ez másképp, vagyis a fogyasztói árat a gyártók döntik el.

A TechARP.com úgy tudja, a PC-gyártók júliustól kínálhatják a Windows 7 frissítésre jogosító utalványokat a Vista-vásárlóknak, akik majd akkor kapják meg a telepítõlemezeket, amikor a Windows 7 valóban megjelenik. A TechARP.com úgy tudja, a Windows Vista Home Premiumot vásárlók Windows 7 Home Premiumra frissíthetnek majd kedvezményesen, Vista Businessrõl Windows 7 Professionalre lehet mjd váltani, Vista Ultimate-rõl pedig értelemszerûen Windows 7 Ultimate-re. Egyelõre nem tudni, pontosan milyen változatokban érkezik a Windows 7, a Microsoft nem hozta nyilvánosságra sem a termékverziókat, sem pedig az árakat.

A Windows 7 nyilvános bétáját várhatóan még a héten bejelenti a Microsoft -- a szoftver már karácsony után kiszivárgott és megjelent a legnagyobb fájlcserélõ hálózatokon. A Las Vegas-i Consumer Electronics Show nyitóelõadását ma este pont Steve Ballmer, a Microsoft elnök-vezérigazgatója tartja, ez az esemény nagyszerû lehetõség lenne a Windows 7 nyilvános bétájának bejelentésére. A Microsoft azonban még mindig adós a legfontosabbal, annak bebizonyításával, hogy a Windows 7 több mint a Vista ráncfelvarrása.


 Azt továbbra sem tudni, hogy az operációs rendszer végleges változata mikor jelenik meg, a Microsoft képviselõi korábban csak annyit árultak el, hogy 3 évvel a Vista premierje után várható a Windows 7, ami 2009 végét vagy 2010 elejét jelenti. A szoftver elsõként angol, német, spanyol, japán és francia nyelven jelenik meg, ezeket nagyjából másfél hónapos csúszással követheti a magyar verzió.
