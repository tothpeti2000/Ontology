Bejelentette a Windows 7 árát a Microsoft

A Microsoft végre nyilvánosságra hozta a Windows 7 különféle változatainak árát. A Home Premium teljes és upgrade változatban is olcsóbb lesz a Windows Vista hasonló változatánál, a Vista Businesst leváltó Professional és az Ultimate pedig ugyanannyiba fog kerülni mint elõdeik. Amint arról már beszámoltunk, a Windows 7 összesen hat változatban jelenik meg, boltban azonban csak a Home Premiumot, a Professionalt és az Ultimate változatot lehet megvásárolni.A Vistához képest fontos változás, hogy a drágább Windows-verziók az olcsóbbak összes képességét tartalmazzák, azaz a Professionalban is lesz Media Center például.
A Home Premium olcsóbb lesz
Emlékeztetõül: a Windows Vista Home Premium teljes dobozos termékként 229 eurós listaáron kapható, az upgrade ára 125 euró. A Vista Business teljes verziójáért 309 eurót, az upgrade-ért 285 eurót kell fizetni, míg az Ultimate-et 319 euróért lehet hazavinni, az upgrade-et pedig 299 euróért. A Windows 7 Professional és Ultimate esetében az árak változatlanok, a Home Premium teljes verziójának ára 199 euróra csökken, az upgrade pedig 119 euró lesz.
Annak érdekében, hogy a cége kedvet csináljon a korai vásárlához, augusztus közepéig a Home Premium és Professional upgrade változatokat különösen alacsony áron kínálja, a Home Premiumot 50 dollárért a Professionalt pedig 100 dollárért, ez azonban csak a kijelölt kereskedõknél és néhány országban lesz elérhetõ (Egyesült Államok, Kanada, Japán, Nagy-Britannia, Franciaország, Németország).
Mivel a vállalat az Európai Unió versenyellenes vizsgálata kihagyja az Internet Explorert a Windows 7 Európában piacra dobott "E" jelû változatából, ezért az upgrade licencet vásárló felhasználók is csak teljes telepítéssel válthatják le korábbi operációs rendszerüket, azaz le kell törölniük a régi Windowst és újratelepíteni a Windows 7-et. Természetesen akik új PC-re elõre telepítve veszik meg a Windows 7-et, azokat ez nem érinti.


A Microsoft várhatóan holnap lebbenti fel a fátylat a Windows 7 Upgrade Option programról, amelynek keretén belül a Windows Vistával telepített PC-k vásárlói ingyen frissíthetnek Windows 7-re, egyedül a telepítõlemez postaköltségét kell kifizetniük. A TechARP.com által kiszivárogtatott információk szerint az akció jövõ áprilisig él, azaz a vistás PC-k vásárlói legkésõbb jövõ tavasszal kérhetik az ingyenes frissítést és persze csak a Vista-változatnak megfelelõ Windows 7 verzióra, vagyis a Vista Home Premiumról Windows 7 Home Premiumra, Businessrõl Professionalre, Ultimate-rõl pedig Ultimate-re lehet váltani -- a Vista Home Basic ebben az akcióban nem vesz részt. A Windows7UpgradeOption.com weboldal szerint a részleteket június 26-án, azaz holnap teszi közzé a Microsoft.
Csak október végén jön a magyar Windows 7
Korábbi információk szerint az október 22-i premierre a magyar kiadás is elkészült volna, a legújabb hírek szerint azonban a magyar vásárlóknak egy hetet várniuk kell a Microsoft legújabb operációs rendszerének lokalizált változatára.A HWSW értesülései szerint a dobozos Windows 7 magyar változata nem október 22-én, hanem egy héttel késõbb, október 31-én kerül a boltokba. Az október 22-i világpremierre csak 14 nyelven készül el az új operációs rendszer (angol, német, spanyol, francia, japán, olasz, holland, orosz, lengyel, brazil portugál, koreai és háromféle kínai), október 31-én egy második hullámban újabb 21 nyelven jelenik meg, köztük csehül, ukránul, svédül, törükül, finnül magyarul is.
