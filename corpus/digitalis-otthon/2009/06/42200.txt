Windows 7 Starter: minden részlet a helyén

A Windows XP és Vista esetében a Starter a fejlõdõ országok piacaira szánt, képességeiben erõteljesen korlátozott verzió volt, a Windows 7 esetében azonban a Starter az egész világon kapható lesz, de a Microsoft továbbra is úgy pozicionálja, mint egy belépõ szintû operációs rendszert. Mivel ez lesz a legolcsóbb változat a Windows 7-bõl, alighanem sok netbook kerül majd forgalomba a Starterrel.
Korábban sok szó esett a Starterrõl, mivel a Windows XP és Vista esetében ez a változat -- a háttérben futó szoftvereken kívül -- csak három program párhuzamos futtatását tette lehetõvé.A Microsoft hivatalos Windows 7 blogján végre megerõsítést nyert az az információ, hogy a Windows 7 Starter nem csak három alkalmazást képes egyszerre futtatni, hanem bármennyit, hasonlóan a többi Windowshoz.
Az egyik szigorú megkötés tehát kikerül a Windows 7-bõl, azonban számos további benne marad. Ezek szerint az operációs rendszer nem támogatja az Aero Glass megjelenítést, ami egyben azt is jelenti, hogy a felhasználóknak le kell mondania az ablakok elõnézetérõl, illetve az Aero Peekrõl. További korlátozás, hogy a Starter felületét egyáltalán nem lehet testre szabni, azaz nem lehet módosítani a háttérképet, az ablakok megjelenését, színét és más tulajdonságait, illetve a hangsémákat sem. Az operációs rendszerben emellett nem lehet kijelentkezés nélkül váltani a felhasználók között.
A Windows 7 Starter multimédiás képességei is csonkák lesznek, a szoftver nem kezel több monitort, kiszedték belõle a Media Centert, valamint a Media Streaming szolgáltatást, amely bárhol a világon lehetõvé teszi a hozzáférést az otthoni gépen tárolt médiafájlokhoz. A DVD-lejátszás is tiltott funkció a Starterben. Mivel az operációs rendszert nem üzleti felhasználásra szánták, ezért a Windows 7 Startert futtató gépeket nem lehet tartományba (domain) léptetni, illetve nem támogatják a Windows XP Mode-ot sem, ami egyébként is csak a Windows 7 Professional és Ultimate verziókhoz letölthetõ extraként érhetõ el.
Azzal, hogy a Microsoft eltörölte a "három párhuzamos alkalmazás" korlátot, alkalmassá tette a Windows 7 Startert a hétköznapi használatra, ugyanakkor továbbra is maradtak benne olyan megkötések, amelyek alighanem sok felhasználót a drágább Windowsok felé terelnek majd. Fõleg a kezelõfelülettel kapcsolatos korlátozások és médialejátszási képességek hiánya lehet az, ami a vevõket a Windows 7 Home Premium vásárlására ösztönözheti. A Microsoft szerint a Windows 7 összes verziója megfelelõ teljesítményt nyújt egy 1,6 GHz-es Atom processzorral és 1 gigabájt memóriával felszerelt netbookon.
