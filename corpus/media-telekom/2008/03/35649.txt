Április bolondja lehet a Windows Mobile 6.1

A Pocket Lint információi szerint április elsején, bolondok napján érkezik meg végre a Windows Mobile operációs rendszer legújabb, 6.1-es változata. Az új szoftverrõl a Las Vegas-i CTIA Wireless konferencián lebbenti majd fel a már nem is olyan sokat takaró fátylat Robbie Bach, a Windows Mobile csapat vezetõje.



A Windows Mobile 6.1 az elsõ jelentõsebb frissítés lesz a Windows Mobile 6.0 tavaly februári debütálása óta. Az operációs rendszerrõl felbukkant különbözõ képek szerint alaposan átdolgozott kezdõképernyõvel, új funkciókkal láthatják majd el a készülékgyártók az okostelefonokat, a megjelenést követõen a közeljövõben. A 6.1-es verzióra frissítés is várhatóan sokkal több készüléknél lesz majd elérhetõ, mint a Windows Mobile 5-öt futtató telefonok, tenyérgépek között.


A korábbi hírek szerint egy Motorola Q9 okostelefonon mutatta be decemberben zárt ajtók mögött a Microsoft az új operációs rendszert. A frissítések között szerepel az Internet Explorer személyre szabható kezdõlapja, a chatszerû SMS-elrendezés, megjelenítés, a Standard változatnál is elérhetõ kivágás-beillesztés funkció, mely a 6.0-nál az érintõképernyõ támogatása miatt hiányzott.

Jelenleg 140 különbözõ termékben használják a redmondiak mobil operációs rendszerét, melyet világszerte 125 mobilszolgáltató is kínál elõfizetõinek. A 2006-os üzleti évben 10 millió Windows Mobile operációs rendszert "szállított" a Microsoft, melybõl 6 millió okostelefonokban kapott helyet. Gyõrffy Péter a Canalys adataira hivatkozva tavaly 65 ezerre becsülte az itthon eladott okostelefonok számát, ebben természetesen nincsenek benne a Windows Mobile-os tenyérgépek -- a "szürkeimporttal" együtt ez jóval magasabb is lehet.
