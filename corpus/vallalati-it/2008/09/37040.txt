A Windows XP tovább él

Az ultraolcsó PC-k kivételével, mint amilyenek a netbookok, a Microsoft 2009. január 31-ével teljesen beszünteti a desktop és médiacenter Windows XP licencek értékesítését, hogy Windows Vista operációs rendszerére terelje át a felhasználókat. Ezt az idõpontot követõen a PC-gyártók teljesértékû számítógépeket nem értékesíthetnek Windows XP-vel.
Windows Embedded, a hosszú távú megoldás
A Windows XP azonban nem csak személyi számítógépeken van jelen, hanem számos megoldásszállító beágyazott eszközökben is alkalmazza: többek között információs kioszkok, ATM-ek, vezérlõrendszerek, értékesítési pontok mûködtetésére. Ez azt jelentené, hogy nekik is az erõforrás-igényesebb és drágább Vista platformra kell áttérniük?
Egyáltalán nem! Beágyazott célú felhasználásra a Microsoft dedikált termékekkel rendelkezik, a Windows Embedded termékcsaláddal, melyek közt a Windows XP több formában is elérhetõ marad, még hosszú, hosszú idõn át - legalább 2016-ig. A beágyazott licencû változatok olyan elõnyökkel is rendelkeznek a desktop verziókhoz képest, mint például az aktiválás mellõzése, ami nagyban megkönnyítheti az eddig asztali verziókkal dolgozók életét, ezenkívül számos üzleti és mûszaki elõnnyel is szolgálhatnak.
Windows XP for Embedded Systems
Azon fejlesztõk és megoldásszállítók számára, akik eddig a Windows XP valamelyik PC-változatát alkalmazták beágyazott platformként, és a lehetõ legkevesebbet akarnak foglalkozni az váltással, azoknak a Windows XP for Embedded Systems változat ajánlott. Ez a kiadás kódszinten tökéletesen megegyezik az asztali Professionallel, eltérés kizárólag a licencelésben mutatkozik, mellyel a Microsoft biztosítja, hogy ez az operációs rendszer nem használható általános célú személyi számítógépekhez - erre utal az XP with Embedded Resctrictions elnevezés, ahogyan más néven emlegetni szokták. Az azonos kódbázis miatt az alkalmazások kompatibilitása garantált az asztali XP-változattal, gyakorlatilag a licenc módosul, mely stabil beszerzési forrást és támogatást jelent hosszú távon is. Új márkastratégiájának részeként a Microsoft bevezette a Windows Embedded Enterprise nevet, mely alatt a Windows XP és a Vista elérhetõ beágyazott licenccel.
Windows XP Embedded
A Windows XP Embedded már egy technikailag is ízig-vérig beágyazott változata az XP-nek, azon megoldásszállítók számára, akik sokkal inkább testreszabott megoldásra vágynak, csökkentve a szoftverréteg erõforrásigényeit, növelni a stabilitást és biztonságot. Az XP Embedded ugyanis komponensekbõl építhetõ fel, mintegy 12 ezerbõl, melyeket a fejlesztõk tetszõlegesen válogatnak össze. Ezzel lehetõvé válik, hogy egy gyakorlatilag teljesen funkcionális XP operációs rendszert kapjunk, mely nagyjából 70 megabájt tárterületet foglal el mindössze. Ebbe a komplett Win32 API, grafikus felhasználói felület, Internet Explorer, Media Player, Terminal Server kliens, USB- és SNMP-támogatás is belefér. Amennyiben ennél kevesebbre van szüksége valakinek, úgy még lejjebb szorítható ez a határ, extrém esetben akár  25 megabájt alá is. Ezzel lehetõvé válik, hogy rendkívül kompakt és alacsony fogyasztású hardver is képes legyen kiszolgálni a rendszert, miközben magasabb szintû stabilitást és biztonságot is kapunk egyúttal.
Az XP Embedded licenc díjazása eltér az Embedded Enterprise árképzésétõl, melynek eredményeként magas számban értékesített eszköz esetében üzletileg is jobban megéri XP Embeddedet választani. Ez a határ nagyjából 20 leszállított eszköznél húzódik meg, e fölött minden egyes újabb eladásnál mintegy50 dollárnak megfelelõ összeggel alacsonyabb a licencdíj. Így nagy mennyiségben akkor is megéri ezt a változatot alkalmazni, ha egy teljesértékû XP Professionalre van szükségünk, ekkor egyszerûen minden komponenst telepíteni kell. Az XP Embedded legalább 2017-ig elérhetõ marad a Microsofttól. Hamarosan, a tervek szerint november elsejétõl piacra kerül az XP Embedded utódja is, az Embedded Standard, mely egy frissített változat - a szigorúbb tesztelés miatt az újdonságok a beágyazott változatokhoz késõbb érkeznek. Az Embedded Standard már .NET Framework 3.5, Windows Server 2008 , Media Player 11 és Internet Explorer 7 támogatást is tartalmaz többek között.
WEPOS
A Microsoft alkalmazásspecifikus változatokat is kínál Embedded családjában. A Windows XP kódbázisára ezekbõl jelenleg az értékesítési pontokra (point-of-sale) szabott Windows Embedded for Point-of-Services (WEPOS) hozzáférhetõ. A WEPOS, és 2009-re várható utódja, a POSReady olcsóbb, könnyebben telepíthetõ, és kevésbé erõforrás-igényes mint egy XP Professional, és legalább 2020-ig támogatott.
Mint látható tehát, a Windows XP kliens és Media Center változatai forgalmazásának beszüntetése egyáltalán nem érinti hátrányosan a beágyazott rendszereket fejlesztõket, sõt. Az eddig hagyományos desktop XP Professionalt alkalmazók számára okot adhat arra, hogy újraértékeljék igényeiket, és a leginkább megfelelõ Embedded változatot válasszák ki, mellyel a hozzáférhetõség és támogatás nyugalma mellett üzleti elõnyhöz is juthatnak a kedvezõbb licencelési feltételek, egyszerûbb telepíthetõség, és az XP Embedded esetében a magas fokú modularitás révén.
A Windows Embedded termékcsaládról október 15-én egy szemináriumot rendez az Elbacom, a Windows Embedded termékek disztribútora a Microsoft Magyarország székhelyén.
A Windows Embedded termékek  az Elbacomnál érhetõek el.
