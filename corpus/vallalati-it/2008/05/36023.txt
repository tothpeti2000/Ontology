Kipróbálhatók az új kisvállalati Windows Serverek

A Microsoft az év második felére ígéri a Windows Server 2008-ra épülõ kis- és középvállalati szervertermékeit, a Windows Small Business Server 2008-at és a Windows Essential Business Server 2008-at, amelyek elõzetes változatait mától próbálhatják ki az érdeklõdõk. Az elõzetes verziókkal együtt a termékek pontos összetevõit, és az árát is bejelentette a Microsoft.



SBS vagy WEBS: mi a különbség?

A Small Business Server (SBS) új változatát az eddigiekhez hasonlóan a legkisebb cégeknek szánja a Microsoft. A Small Business Server 2008 része az Exchange Server 2007 Standard, a SharePoint Services 3.0, a Forefront Security for Exchange Server biztonsági csomag egy éves elõfizetéssel. Az SBS 2008 prémium változata emellett tartalmaz egy második Windows Server 2008 licencet, illetve az SQL Server 2008 Standard változatát, amelyet akár második szerverre is telepíthetnek a felhasználók.

Az 50-250 PC-vel rendelkezõ középvállalatoknak tervezett Windows Essential Business Server 2008 Standard a Windows Server 2008 mellett tartalmazni fogja az Exchange Server 2007-et, a Forefront Security for Exchange 2007-et, a System Center Essentials 2007-et, valamint a Threat Management Gatewayt (leánykori nevén ISA Server). A WEBS Premium verziójában helyet kap még az SQL Server 2008 is.

Az SBS és a WEBS közötti jelentõs különbség a komponenseken kívül, hogy az SBS 2008 felületét úgy alakították ki, hogy a lehetõ legegyszerûbben telepíthetõ és kezelhetõ legyen, ugyanis a célközönségnek számító kisvállalatoknál az informatikai kompetencia gyakran korlátozott. Az SBS 2008 ezért olyan központi menedzsment-konzolt kapott, amelyrõl minden funkció egyszerûen elérhetõ, akár távolról is. A WEBS már átfogó(bb) felügyeleti lehetõséggel rendelkezik, és ebbe a felületbe már külsõ gyártóktól származó modulok is integrálhatók.

Mennyibe fog kerülni?

A Windows Small Business Server 2008 Standard ára öt klienslicenccel (CAL) 1089 dollár, a további klienslicencek 77 dollárért vásárolhatók meg. Az SQL Server 2008-at is tartalmazó Small Business Server 2008 Premium öt klienslicenccel 1899 dollárba kerül, minde további CAL ára 189 dollár. A Windows Essential Business Server 2008 Standard öt CAL-lal 5472 dollárba kerül, a további CAL-ok ára 81 dollár, a WEBS Premium 7163 dollárba kerül, minden további CAL 195 dollár. Ezek az árak a "dobozos" verziókra vonatkoznak, az OEM- és mennyiségi licencvásárlók olcsóbban juthatnak a termékekhez. A hazai árakat egyelõre nem tudni, de a HWSW információi szerint igazodni fognak a kintiekhez.


 Licencelési szempontból jelentõs újdonságnak számít, hogy az ügyfelek már nem csak csomagban, hanem egyenként is vásárolhatnak klienslicenceket (CAL). Ezek a licencek a hálózaton található más Windows Server, SQL Server és Exchange Server termékekre is érvényesek, így azokhoz nem szükséges még további licenceket beszerezni. Itt lehet érdemes megjegyezni, hogy azok a cégek, amelyek már használnak Windows Servert vagy Exchange Servert, kedvezményesen térhetnek majd át Windows Essentials Business Serverre.

"Ezzel a két csomaggal a Microsoft komplett megoldásokat kínál olyan cégeknek, ahol nincs belsõ informatika és csak külsõ partnerekre támaszkodnak (10-50 PC) és olyanoknak, ahol csak kevés belsõ informatikus van, de nekik mindenhez értenie kell" -- mondta el a HWSW-nek Keszei Balázs, a Microsoft hazai leányvállalatának termékmenedzsere, aki hozzátette, hogy hazánkban az ügyfelek és a partnerek részérõl egyaránt élénk az érdeklõdés az SBS 2008 és a WEBS 2008 iránt. A magyar nyelvi változatok az angol után mintegy 2 hónappal jelennekg majd meg -- tudatta lapunkkal Keszei.
