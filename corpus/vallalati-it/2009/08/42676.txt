Megjelent a Virtual PC és a Windows XP Mode kiadásra szánt változata

Az új asztali operációs rendszer Professional, Enterprise és Ultimate változatára utólag letölthetõ Windows XP Mode egy elõre telepített Windows XP-t tartalmazó virtuális gép, amely lehetõvé teszi a Windows XP-re írt alkalmazások változtatás nélküli futtatását Windows 7-en. A Microsoft ezt egyfajta "végsõ menedéknek" építette a Windows 7-be annak érdekében, hogy ha egy XP-re írt alkalmazást semmilyen más módszerrel nem lehet Windows 7-re portolni, a kompatibilitási konfliktus ne gátolja meg az operációs rendszer cseréjét.
Amint az ismert, a Windows 7 a Vista továbbfejlesztett kernelére épül, így a Vistára írt szoftverek biztosan futnak az új operációs rendszeren. A Vista azonban nagyon sok olyan változtatást hozott a Windows XP-hez képest (pl. User Account Control), amely akár lehetetlenné is teheti a Windows XP-re (rosszul) megírt alkalmazások használatát. A Microsoft ugyan számos olyan eszközt bocsát a fejlesztõk rendelkezésére, amelyek segítségével tesztelhetik programjaik kompatibilitását és javíthatják az esetleges hibákat, néha ez sem vezet célra, például ha a fejlesztõ és a kód már nem elérhetõ. Ilyenkor segíthet a Windows XP Mode.

Windows XP Mode-ban az alkalmazást ugyan a virtualizált Windows XP-re kell telepíteni, de a gazda operációs rendszerbõl, azaz a Windows 7-bõl ugyanúgy indítható mint bármilyen más program. Így a felhasználók egyetlen asztali környezetbõl futtathatják a Windows XP-t igénylõ alkalmazásaikat a Windows 7-en natívan mûködõ programokkal.
Az ingyenesen letölthetõ kiegészítõ a következõ generációs Virtual PC-re épül, amelynek szintén elérhetõvé vált a megjelenésre szánt változata. Az XP Mode elõnye a VMware-en vagy más hypervisoron virtualizált Windows XP-vel szemben, hogy az XPM mûködése központilag adminisztrálható Active Directory és csoportházirendek segítségével. Emellett az sem elhanyagolható tényezõ, hogy az XPM önmagában tartalmaz egy teljes értékû Windows XP SP3 licencet, így azért nem kell külön fizetni.
Az XP Mode elsõ elõzetesét még áprilisban adta ki a Microsoft. Az RC számos újdonságot tartalmaz, például támogatja a Windows 7 taskbar gyorsmenüit, vagyis a virtualizált XP-re telepített alkalmazások innen is elérhetõk. További figyelemre méltó fejlesztés, hogy a virtualizált XP-hez közvetlenül a Windows 7-bõl lehet USB-s eszközöket csatlakoztatni, kikapcsolható a Windows XP és Windows 7 közti meghajtómegosztás, emellett a szoftverbe bekerült egy gyorstalpaló használati utasítás, amely az elsõ indításkor elmagyarázza a Windows XP Mode mûködését.
A Windows XP Mode RC letölthetõ a Microsoft weboldaláról. A szoftver futtatásához Windows 7 Professional, Enterprise vagy Ultimate RC vagy RTM szükséges. A hardverkövetelmények: hardveres virtualizációs támogatással rendelkezõ processzor, 1 gigabájt szabad memória a virtuális gép részére, valamint 15 gigabájt terület a merevlemezen.
