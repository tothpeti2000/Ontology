Hivatalos: Európába is Internet Explorerrel jön a Windows 7

Amint arról lapunk is beszámolt, a cég egy héttel ezelõtt felajánlotta az Európai Bizottságnak, hogy az õsszel érkezõ új operációs rendszerben a telepítés után lehetõvé teszi a felhasználók számára hogy az Internet Explorer helyett más böngészõt válasszanak és akár véglegesen el is távolítsák az IE-t a rendszerbõl. A választás lehetõsége a Vista- és XP-felhasználók számára is adott lesz egy automatikus frissítés révén. A Microsoft abban bízik, a Bizottság elégedett lesz ezzel a lépéssel és nem bünteti meg versenyjogsértõ magatartás miatt.
Bár a Bizottság hivatalosan egyelõre nem adta áldását erre a lépésre, a Microsoft az eddig kapott visszajelzések alapján mégis úgy határozott, hogy nem dobja piacra az Internet Explorer nélküli Windows 7E változatot és Európában is a "teljes" Windows 7-et kapják meg a felhasználók. "A múlt heti fejlemények fényében, illetve a Windows 7E-re a PC-gyártóktól érkezett visszajelzések tükrében örömmel jelenthetem be, hogy októberben Európában is ugyanazt a Windows 7-verziót szállítjuk majd, mint bárhol máshol a világon" -- írta egy blogbejegyzésben Dave Heiner, a redmondiak jogásza.

A PC-gyártók szerint igen bonyolult helyzet állt volna elõ, ha gépeikre a böngészõ nélküli Windows 7E-t telepítik, majd késõbb, esetleg az operációs rendszer premierje után az Európai Bizottság elfogadja a Microsoft által javasolt böngészõválasztó képernyõt. Ez a gyártóknak is problémás helyzet, mivel a gépeiken le kell cserélni az operációs rendszert, másrészt a felhasználók számára is zavaros lett volna, hogy melyik verziót is kapják.
A Windows 7E elvetése egyben azt is jelenti, hogy az európai felhasználók újratelepítés nélkül tudnak Windows Vistáról Windows 7-re upgrade-elni. Windows 7E-vel ez lehetetlen lett volna, az Internet Explorer hiánya miatt az operációs rendszer cseréje csak teljes újratelepítessel lett volna kivitelezhetõ. A Windows 7E-t elõrendelõ vásárlóknak a teljes, Internet Explorert is tartalmazó Windows 7-változatot fogja szállítani a Microsoft.
