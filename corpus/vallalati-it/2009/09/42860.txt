Letölthetõ már a Hyper-V Server 2008 R2

A Hypver-V Server 2008 R2 egy önálló hypervisor, praktikusan egy célra szabott Server 2008 R2 Enterprise Edition "Core telepítés", mely operációs rendszerek széles skáláját képes vendégként fogadni. A Microsofttól szabadon letölthetõ szoftver jelentõsen továbblépet elõdjén, így többek közt már jóval nagyobb gazdagépeket is ki tud használni a nagyobb arányú szerverkonszolidáció támogatása érdekében. Az R2 8 foglalatot, és 64 szálat kezel, amit 384 virtuális gép közt tud szétosztani, amihez 1 terabájt fizikai memóriát tud megcímezni - ezek virtuális gépek azonban legfeljebb 512 virtuális processzort használhatnak, vagyis nem lehet mindegyik SMP-ben.

A fontos újdonságok közé tartozik a gazdaszintû fürtözés a magas rendelkezésre állású (HA) virtuális környezetek kialakítása érdekében, a teljes hibatûrés egyelõre nem valósult meg, igaz ilyen képességekkel a VMware is csak korlátozottan, uniprocesszoros vendégek esetében rendelkezik. Szintén mérföldkõ az élõ migráció támogatása, valamint a processzorerõforrások dinamikus mikroparticionálása, vagyis több VM egy processzormagra történõ online konszolidációja, ami azt is lehetõvé teszi, hogy csak a szükséges számú processzormagot használjuk (Core Parking).
Szintén adatközpont szintû követelmény a virtuális hot-swap SCSI meghajtók támogatása, vagy a virtuális processzorok tehermentesítése a TCP-hálózati forgalom alól, valamint a Jumbo Frame-ek támogatása, ami együttesen drasztikusan megnöveli a virtuális gépek hálózati teljesítményét. A támogatott vendégrendszerek listáját itt tekinthetjük meg. A Hyper-V képességei azonban csak Windows Server környezetben teljesednek ki, amire például fürtözéskor és központi menedzseléskor a tartományba léptetés érdekében van szükség, de lokális grafikus felületet, addicionális szerepkörök beállítását is csak a Windows Server 2008 R2-vel együtt kapunk, melyet augusztus óta lehet kipróbálni - grafikus felületet persze távoli eléréssel, a Windows 7 Remote Server Administration Tools (RSAT) révén is szerezhetünk.
Mindez azonban nem jelenti azt, hogy hogy a Hyper-V Server 2008 R2 képességei korlátozottak lennének a Windows Server 2008 R2 with Hyper-V változathoz képest, a hypervisor ugyanaz. A Hyper-V Server éppúgy része lehet egy nagyvállalati rendszernek, mint a Windows Server. Éppúgy felügyelheti a System Center Virtual Machine Manager, a System Center Operation Manager, vagy mentheti a Data Protection Manager. Lehet tartománytag, érvényre juthat rá a csoportházirend, és így tovább. A Hyper-V Server ráadásul USB-rõl is bootolható, ami nagy segítség lehet egy szerverinstallációkor.
A pontosításokért köszönet illeti Lepenye Tamást, aki a Microsoft Magyarország nagyvállalati üzletágánál a virtualizációért és rendszerfelügyelért felelõs rendszermérnök. Lepenye blogbejegyzését a Hyper-V Server 2008 R2-rõl itt lehet olvasni.
