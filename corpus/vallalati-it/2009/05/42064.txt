Elérhetõ a Visual Studio 2010 és a .NET 4.0 bétája

Tegnap óta letölthetõ a Visual Studio 2010 és .NET Framework bétái a Microsoft Developer Networkrõl, és a tervek szerint holnaptól teljesen publikusan is elérhetõ lesz. A vállalat szerint a a béták teljesítménye sokat javult a tavalyi CTP (Community Technology Preview) óta. 
A Visual Studio 2010 számos ponton megújul elõdeihez képest. A vállalat azt ígéri, letisztultabb felületet, egyszerûsített kezelhetõséget ígér a továbbfejlesztett kódszerkesztõ mellett. A kiemelt változások közé tartozik a SharePoint fejlesztések sokkal magasabb szintû támogatása, ami a testreszabhatóságot és egyszerûbb tesztelést igyekszik takarni. Ezen túlmenõen kiemelt szempontot kapott a webes fejlesztésekhez az erõteljes JavaScript motor és a Silverlight teljeskörû támogatása.&nbsp;
További továbblépést jelent a Windows 7-hez és a Microsoft Azure cloudjához érkezõ eszközök, valamint az IBM DB2, az Oracle és a Microsoft SQL Serverekkel való együttmûködés DSP-k (Database Schema Provider) révén. Az IBM már elkötelezte magát DSP-k készítésére, értékesítésére és támogatására. A vállalat kiemelte, hogy a Visual Studio Team System 2010-es kiadásával demokratizálni akarja az alkalmazások életcikulásnak menedzselését, vagyis segíti az érdekelt fejlesztõk, projektvezetõk, adatbázis szakemberek és tesztelõk kollaborációját.
A .NET keretrendszer következõ generációjának képességei egyelõre a háttérben maradtak, a vállalat a funkcionalitás jelentõs növelésérõl beszél a Windows Communication Foundation és Windows Workflow Foundation estében. Ezek többek közt új REST kommunikációs képességeket, munkafolyamat modelleket és új vizuális tervezõt is takarnak. Az új .NET és Visual Studio nem rendelkezik publikus megjelenési dátummal.
