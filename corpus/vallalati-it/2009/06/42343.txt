Mégsem kizárt, hogy 2011-ig lehet XP-re downgradelni Windows 7-rõl

Az eredeti tervek szerint a Windows 7 Professional és Ultimate változatról a felhasználók visszatérhettek volna Windows XP-re, azonban ez a lehetõség csak az új operációs rendszer megjelenésétõl számított 6 hónapig állt volna rendelkezésre, ezt követõen csak Vistára lehetett volna downgrade-elni. A Windows 7 világszerte október 22-én kerül forgalomba, ezek szerint a vásárlók 2010 áprilisának végéig válthatnak vissza Windows XP-re.
A cégek ezért ha nem akartak volna kétféle operációs rendszert üzemeltetni, a Windows 7 megjelenését követõ 6 hónapban kellett volna megvásárolniuk az összes PC-t, amit a Windows 7 bevezetése elõtt használni szeretnének, vagy olyan mennyiségi licencszerzõdést kellett volna kötniük a Microsofttal, amely megengedi a downgrade-et. További megoldást jelenthet a számítógépeket Windows Vista Business-szel vagy Ultimate-tel vásárolni, majd downgrade-elni Windows XP-re, ezt késõbb upgrade licenccel lehet Windows 7-re cserélni. Windows XP-rõl azonban közvetlenül nem lehet Windows 7-re frissíteni, csak Vistáról -- a helyzet összességében szoftverlicencelési rémálmot jelenthetett volna egyes vállalatok számára.
A Microsoftnál a jelek szerint utólag rájöttek, hogy a licencelési rendszer ebben a formában komoly buktatókat rejt magában, így órákkal azután, hogy neves elemzõk élesen bírálták az új downgrade-gyakorlatot, változtattak a feltételeken. A ComputerWorld megkeresésére a Microsoft egyik szóvivõje e-mailben elmondta, hogy az új feltételek szerint a Windows 7 Professional és Ultimate felhasználók legfeljebb nem 6, hanem 18 hónappal a Windows 7 megjelenése után kérhetnek downgrade-et. Ez az intervallum abban az esetben lehet csak rövidebb, ha ennél elõbb jelenik meg az operációs rendszer elsõ javítócsomagja.


Elemzõk szerint mindez összességében átgondoltabb licencpolitikáról árulkodik, Michael Silver, a Gartner szakértõje azonban úgy látja, szerencsésebb lett volna megfordítani a feltételeket és a késõbb bekövetkezõ eseményhez szabni az XP downgrade határidejét (azaz legalább a Windows 7 megjelenésétõl számított 18 hónap, legkésõbb az SP1 megjelenési dátuma). A jelenlegi rendszerrel bár nem kizárt, hogy a downgrade-re még 2011-ben is lehetõség lesz, valójában ennek kicsi a valószínûsége, az eddigi gyakorlat alapján jóval valószínûbb ugyanis, hogy a Windows 7 SP1 még jövõre elkészül.
A Windows Vista elsõ javítócsomagja az operációs rendszer megjelenését követõ 14 hónapon belül érkezett meg, a Windows XP elsõ javítócsomagjára csak tíz hónapot kellett várni, míg a Windows 2000 SP1 csupán fél évvel követte az operációs rendszert.
