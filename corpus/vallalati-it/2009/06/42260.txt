A magyar vállalatok már várják a Windows 7-et

Több mint ezer rendszergazda, informatikai döntéshozó, fejlesztõ és más szakember vett részt a Microsoft TechNet rendezvényen, amelyrõl közel 600-an nyilvánítottak véleményt egy kérdõív formájában, amelyek eredményei alapján a hazai cégek jellemzõen várják a Windows 7-et és a Windows Server 2008 R2 érkezését.
A nem reprezentatív felmérés szerint a hazai vállalatoknál a legnépszerûbb vállalati asztali operációs rendszer továbbra is a Windows XP, a válaszadók több mint háromnegyede ezt jelölte meg mint elsõdlegesen használt szoftvet. Noha a Vista már több mint két éve piacon van, csupán a cégek hatoda használja elsõdleges asztali operációs rendszerként. 
A szerverek esetében is az XP-generációhoz tartozó Windows Server 2003 a legnépszerûbb, a hazai cégek közel kétharmadánál használják. A Windows Server 2008 eddigi rövid élete alatt majdnem ugyanakkora részesedést tudott elérni mint a Linux, mindkettõt a vállalatok ötödénél teljesít szolgálatot. A negyedik legelterjedtebb szerver operációs rendszer a Windows 2000 -- derül ki a felmérésbõl, amely szerint a hazai cégek közel négyötödénél a Windows Server valamelyik verziója a preferált szerverplatform.
A TechNet résztvevõi közül sokan próbálták ki a Windows 7 és a Windows Server 2008 R2 kiadásra jelölt (RC) változatát, amely május elején jelent meg. Valószínûleg az ezzel szerzett kedvezõ tapasztalatoknak, a Vista rossz hírének, illetve a Windows XP elöregedésének köszönhetõen a hazai cégek kimondottan várják a Windows 7-et és a Windows Server 2008 R2 érkezését, a válaszadók 6 százaléka fél éven belül, további 20 százaléka pedig egy éven belül be kívánja vezetni õket -- a maga idejében a Vista esetében jóval alacsonyabbak voltak ezek az értékek.
A Microsoft új operációs rendszereinek legfontosabb újdonságait bemutató elõadások és videók a héten kerülnek fel a TechNet weboldalra, ahonnan bárki letöltheti õket.
