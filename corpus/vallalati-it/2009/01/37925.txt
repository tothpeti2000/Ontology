Februári Informatika Tisztán: Exchange Server 2007

Sok vállalat kommunikációjának lelke, az Exchange Server 2007 lesz az Informatika Tisztán programsorozat következõ témája. A februári állomáson egy teljes napot kap a szoftver, így nem csak a korábbiakhoz hasonlóan érintõlegesen, hanem mélységeiben tárgyalják majd az elõadók. 



A február 17-én Budapesten megrendezett, ingyenes Informatika Tisztán napon három MCT, vagyis Microsoft Certified Trainer fogja várni a látogatókat, elsõsorban rendszergazdákat és üzemeltetésért felelõs szakembereket. A rendezvényen szó esik majd az Exchange Server 2007 felépítésérõl, beleértve a szerepkörök jelentõségét és típusaikat, a protokollokat, valamint a biztonsági képességeket. Ezenkívül szó esik az Exchange Server 2003-ról történõ migráció mikéntjérõl, valamint a szabályok használatáról is, melyekkel a levelek forgalmát irányíthatjuk. Jelentkezés és további részletek a TechNet oldalán. 


Ahogyan a többi Informatika Tisztán rendezvény, úgy az Exchange Server 2007 nap elõadásai is szabadon letölthetõvé válnak utólag. Novemberben például a windowsos hálózatok biztonságát tárgyaló elõadásokat tette elérhetõvé a Microsoft, beleértve a videofelvételeket is. A tavaly õsszel elindított Informatika Tisztán rendezvénysorozat célja az, hogy a gyakorló rendszergazdák, illetve rendszergazdának készülõ diákok naprakész ismereteket kapjanak a Microsoft legújabb, de már piacon levõ technológiáiból, így a megszerzett tudás azonnal kamatoztatható.
