IBM szoftverek az Amazon cloudjában

A két vállalat közt megköttetett megállapodás értelmében elérhetõek lesznek az Amazon cloudján az IBM egyes vállalati szoftverei. A konstrukcióhoz nem csak új licenccel, hanem meglévõ felhasználóként is hozzá lehet férni majd a csúcsidõszakok terhelésének kiszolgálására. Az IBM ezzel követi legnagyobb szoftverriválisait, az Oracle-t, a Microsoftot és a Sun Microsystemst, melyek már megjelentek az Amazon cloudjában.



Az IBM és az Amazon Web Services elsõ körben az IBM DB2, Informix Dynamic Server ,WebShpere Portal, Lotus Web Content Management, WebShpere sMash szoftvereket, valamint az azoknak otthont adó Novell SUSE Linux operációs rendszert teszi elérhetõvé az Elastic Compute Cloudban (EC2), mely egyébként Xen hypervisort alkalmazó x86-os Linux-rendszereken fut.  

Az IBM a cloudon keresztül ingyenesen hozzáférhetõ Amazon Machine Image-eket (AMI), vagyis elõre felkészített rendszerképeket kínál az alkalmazások cloudba mozgatásának teszteléséhez, az EC2-ben felhasznált erõforrásokért azonban természetesen fizetnünk kell. A teszteléshez a DB2 Express-C 9.5 és Informix Dynamic Server Developer Edition 11.5 adatbázisok, a WebSphere Portal Server és Lotus Web Content Management Standard Edition portálmotor és tartalomkezelõ rendszer, továbbá a WebSphere sMash webes fejlesztõ és futtatókörnyezet férhetõek hozzá.


A végleges, éles rendszerben használható és támogatott rendszerképek bétája a következõ hónapok során érkezik majd, ahogyan a késõbbi képek árazása is kialakulóban van még. Az EC2 cloudban futó IBM szoftverek licencének árazása lényegében megegyezik a vállalat x86-os fizikai gépekre adott licenceivel, logikai processzoronként 50 PVU, vagyis Processor Value Unit. Jelenleg legfeljebb 8 virtuális processzormaghoz juthat egy vendéggép, ez a számítási kapacitás pedig óránként 205 forintba kerül, vagyis egy egész évre 1,8 millió forintra rúgna.

Az óránkénti felbontásnak azonban éppen az a lényege, hogy annyi kapacitást vásároljunk, amennyire éppen szükségünk van. Lehetõség nyílik akár egytlen processzorra (egy nagyjából 1 gigahertzes Intel Core magnak felel meg) és 1,7 gigabájt memóriára visszaskálázni az erõforrásokat, amiért mindössze 25 forintot kell fizetnünk óránként, vagyis évi 220 ezer forintot az EC2-nél üzemelõ virtuális gépünk alja -- Linux esetében, a Windows 25 százalékos felárat jelent. Jelenleg Red Hat Enterprise, Oracle Enterprise, OpenSUSE, Ubuntu, Fedora, Debian és Gentoo Linux disztribúciók érhetõek el az EC2-ben, továbbá OpenSolaris és Windows Server 2003. A számítási és memóriakapacitáson túl a hálózati forgalomért és az adatterületért is fizetnünk kell -- a tárterületért havi 25 forintot gigabájtonként és ugyanennyit egymillió lemezmûveletért, továbbá 23 forintot minden beérkezõ gigabájt, és 40 forintot minden kimenõ gigabájt után.

Az IBM nem az elsõ, mely a szoftvereit felkészíti az Amazon EC2-ben való mûködésre. Az Oracle 11g, a Microsoft SQL Server Standard 2005, és a Sun MySQL Enterprise adatbáziskezelõk már jelen vannak, és testreszabható AMI-kat lehet belõlük létrehozni, ahogyan elõre elkészített AMI-k százai állnak rendelkezésre. Hozzáférhetõ Apache és IIS is, ahogyan Java Application Server és JBoss Enterprise Application Platform, vagy Ruby on Rails is. Látható, hogy az Amazon jelenleg elsõsorban a webes alkalmazásokra helyezi a hangsúlyt, ami a cloud természetébõl fakadóan kézenfekvõ. További információ, leírások és útmutatók az IBM-Amazon cloud kezdeményezésérõl az IBM honlapján olvashatóak.
