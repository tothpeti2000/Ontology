Feltörték a Windows 7 aktivációs rutinját

A Lenovótól kiszivárgott Windows 7 OEM elõször kínai oldalakon tûnt fel, majd világszerte elterjedt, ebbõl nyerték ki kalózok azokat a kódokat (az OEM-SLP kulcsot és az OEM aktivációs tanúsítványt), amelyeket felhasználva aktiválni tudták az internetre kiszivárgott végleges, 7600.16358 buildszámú Windows 7 Ultimate-et. A Microsoft azóta megerõsítette az ügyrõl beszámoló Neowin.net oldalnak, hogy tudomása van a Windows 7 aktivációs rendszerének megtörésével kapcsolatos próbálkozásokról. A vállalat határozottan leszögezte, hogy a Windows 7 letöltése fájlcserélõ hálózatokról kalózkodásnak számít és ezzel a felhasználó biztonsági kockázatoknak teszi ki magát.

Ami igaz az igaz, a fórumokban olvasható beszámolók szerint a Windows 7 interneten terjedõ különféle elõzeteseiben már többször felbukkantak kártékony kódok. Az ilyen kockázatok kivédésére már egy alkalmazás is készült, a Windows 7 ISO Verifier neve mindent elárul, amit csak tudni kell róla: megnézi, legitim forrásból származik-e a letöltött Windows 7 lemezkép. Az ISO Verifier nem mást csinál, mint a letöltött .ISO állományról készít egy MD5 hash-t, és ezt összeveti a már ellenõrzött lemezképekrõl készített, tehát megbízható forrás alapján készül hashekkel.
A Windows 7 végleges változatának elkészültét július 23-án jelentette be a Microsoft. A szoftverfejlesztõk és a hardvercégek, illetve a TechNet-elõfizetõk augusztus 6-án kapján meg a szoftver végleges kódját, amelyet a Microsoft Connectrõl vagy az MSDN-rõl tölthetnek le. A Microsoft Gold és Minõsített (Certified) partnerei augusztus 16-án érhetik el az új operációs rendszeret, de csak angol nyelven, más nyelveken csak októbertõl. A mennyiségi licencszerzõdéssel és élõ Software Assurance megállapodással rendelkezõ üzleti ügyfelek és az MSDN-elõfizetõk augusztus 7-én már letölthetik a végleges kódokat, a Software Assurance nélküli mennyiségi licencvásárlók számára szeptember 1-én érkezik el a nagy nap. A kiskereskedelmi megjelenés világszerte október 22-én lesz, Magyarországon pedig október 31-én.
Aki ki szeretné próbálni a Windows 7 kiadásra jelölt változatát, amely a Microsoft tájékoztatása szerint nem sokban különbözik a végleges kódtól, augusztus 20-ig töltheti le a Microsoft weboldaláról. A termékhez aktivációs kulcsokat még késõbb is kiad a szoftvercég, de letölteni ezután (legális forrásból) már nem lehet a Windows 7-et. A feltelepített RC jövõ nyárig használható.
Egyelõre nem tudni, a Microsoft hogyan védi ki a most kikerült  aktivációs trükköt, amelyhez egyébként némi BIOS-módosításra is szükség van. Annak idején a Windows Vista is hamar elesett a kalózokkal folytatott harcban, azonban az SP1-ben a cég blokkolta a legnépszerûbb aktivációs trükköket.
