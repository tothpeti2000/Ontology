A nagyvállalati felhasználók már nyáron megkapják a Windows 7-et

Egyes mennyiségi licencvásárlók már a nyáron megkaphatják a Windows 7 végleges változatát -- olvasható az All About Microsoft blogban, amelyet a ZDNet egyik újságírója, Mary-Jo Foley vezet. A redmondi vállalattal foglalkozó blog szerint a cég július 10-én értesítette partnereit arról, hogy az augusztus 31-ig Software Assurance konstrukcióban mennyiségi licencet vásárló ügyfelek augusztus 1-tõl jogosultak a végleges Windows 7 Enterprise használatára a PC-iken.
A Microsoft a Windows Vista megjelenésekor is hasonló megoldást választott, a mennyiségi licencvásárló vállalati ügyfelek jóval a hivatalos kiskereskedelmi rajt elõtt már használhatták a szoftvert. A Microsoft korábbi bejelentése szerint a Windows 7 október 22-én kerül kereskedelmi forgalomba, Magyarországon pedig október 31-én mutatkozik be.
Kész a végleges kód?
A világ legnagyobb szoftvercége a múlt héten nevezte ki a Windows 7 fejlesztéséért felelõs Steven Sinofskyt a Windows-üzletág élére és egyben megerõsítette, az új operációs rendszer végleges kódja még júliusban gyártásba megy (RTM). A magyar Windows 7 blog szerint a végleges, 7600-as build már kiszivárgott az internetre, és felbukkant a fájlcserélõkön. Ez a verzió már nem mûködik a bétához és RC-hez kiadott kulcsokkal és legfeljebb 30 napig használható, mint ahogy a kereskedelmi forgalomba kerülõ operációs rendszer is, amelyet a telepítéstõl számított 30 napig lehet aktiválni. A Microsoft valószínûleg a ma kezdõdõ Worldwide Partners Conference rendezvényén jelenti be hivatalosan, hogy elkészült a szoftver. (Olvasónk jelezte, hogy a kiszivárgot 7600-as build nem a végleges, bõvebb információ a hozzászólásokban.)

A Windows 7 a Vistához hasonlóan számos változatban elérhetõ lesz azonban a Microsoft szerint a Home Premium, a Professional és az Ultimate változat lefedi szinte az összes ügyféligényt, így a cég ezek népszerûsítésére koncentrál majd. Ezek mellett elérhetõ lesz a Home Basic változat (a fejlõdõ piacokra), a nagyvállalatoknak szánt, csak mennyiségi licencben kapható Enterprise, illetve a netbookokra szánt Starter. Az egyes verziók közti különbségekrõl a Windows 7 blog ad kimerítõ tájékoztatást.
A Windows 7-tel együtt természetesen a szerverváltozat, a Windows Server 2008 R2 is elkészül, ennek kódját szintén július végén zárja le a Microsoft. A Windows Server 2008 R2 kereskedelmi bevezetése egybeesik a Windows 7-tel, ez is október 22-tõl lesz kapható, Magyarországon pedig október 31-én jelenik meg.
Tízbõl hat cég kihagyná?
A windowsos felügyeleti szoftvereket fejlesztõ ScriptLogic több mint 1100 válaszadó bevonásával készített felmérése szerint a cégek közel kétharmada egyelõre nem tervezi a Windows 7 bevezetését. A megkérdezettek 5 százaléka még az idén áttér az új operációs rendszerre, harmaduk pedig 2010 végéig fogja bevezetni az új operációs rendszert. Az áttérés legfõbb kerékkötõje az idõ- és erõforráshiány, amelyet a megkérdezettek 42 százaléka nevezett meg, a második legfontosabb pedig az alkalmazáskompatibilitás. A felmérés szerint a cégek 37 százaléka véli úgy, hogy az új szoftverbevezetések elhalasztásával vagy éppen kihagyásával jelentõs költséget tud megtakarítani.
